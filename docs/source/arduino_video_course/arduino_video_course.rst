Arduino ビデオコース
======================

SunFounder の 3 in 1 スターターキットを使用して、Arduino の世界を広く深く学ぶことができる Arduino ビデオコースにご参加ください。このシリーズは、Arduino エコシステムと uno R3 ボードの機能についての紹介から始まり、実践的な応用とプログラミング技術への徹底的な探究の土台を築きます。LED の制御の基礎、シリアル通信の理解、RGB LED、ボタン、シフトレジスタなど様々なコンポーネントの操作方法を学びます。また、スマートカープロジェクトやIoTプロジェクトなど、多彩で楽しいプロジェクトも提供します。このコースを一歩一歩進めることで、単にコードをコピー＆ペーストするのではなく、自分自身でコードを書き、好みに合わせてArduinoプロジェクトを実装するスキルを身につけることができます。
