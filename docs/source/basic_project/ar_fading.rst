2.1 フェーディング
=====================

このプロジェクトは :ref:`ar_blink` に似ていますが、違いは信号の種類です。
前者はデジタル信号（0&1）を出力してLEDを点灯または消灯させるのに対し、このプロジェクトはアナログ信号を出力してLEDの明るさを制御します。
