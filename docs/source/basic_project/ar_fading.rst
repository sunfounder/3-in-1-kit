2.1 フェージング
=================

このプロジェクトは :ref:`ar_blink` に似ていますが、信号の種類が異なります。
前者はデジタル信号（0&1）を出力してLEDの点灯/消灯を制御するのに対し、このプロジェクトはアナログ信号を出力してLEDの明るさを調節します。
